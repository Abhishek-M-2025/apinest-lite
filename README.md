# 🚀 APINest Lite

A lightweight browser-based API Testing Platform built with **Vanilla JavaScript** and **Node.js + Express**.

## ✨ Features

- 🌐 GET, POST, PUT, PATCH, DELETE, QUERY
- 📌 Query Params & Headers
- 📦 JSON, Raw, Form Data, URL Encoded
- 🔐 No Auth, Bearer, Basic, API Key
- 📄 Response Viewer (Status, Headers, Time, Size)
- 📂 Collections
- 🌍 Environment Variables
- 🕘 Request History
- 📋 Copy Response
- 💾 LocalStorage Support

---

## 🛠 Tech Stack

## 🔄 Application Flow

```text
User
  │
  ▼
Fill API Details
(URL, Method, Headers, Body)
  │
  ▼
Click Send
  │
  ▼
Express Proxy Server
  │
  ▼
Target API
  │
  ▼
Receive Response
  │
  ▼
Display Status, Headers,
Time, Size & Body
  │
  ▼
Save to History /
Collections (LocalStorage)
```
